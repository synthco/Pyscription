# Pyscrption 

Генератор README.md для Python 

---
## Проблематика
Створення README файлів для Python-проєктів - ручна, не цікава робота, яка може забирати багато часу. 

Використання LLM для цього в лоб є дорогим задоволенням, оскільки вимагає насдислання всього коду проєкту, що може споживати тисячі токенів. 

## Рішення 
CLI - інструмент, який генерує README, дотримуючись принципу студентської (радикальної) економії

---
## Архітектура 
1. Rust-компонент парсить код і локально генерує детерміновану частину документації. API Reference (спиосок класів, методів, функції та їхніх докстрінгів)
2. Клієнт Gemini викликається тільки для написання секцій Intoduction та Usage Example. Замість всього коду йому надсилається тільки дострінг модуля вищого рівня, а це короткий промпт 

## Очікуваний результат
Користувач отримує повний RAEDME, витративши приблизно на 80% менше токенів.
